 **FUTURE_CS_02**

 **Phishing Email Detection & Awareness Report**

This repository contains a phishing email detection and awareness report prepared as part of the Future Interns Cyber Security Internship.

 **Objective**

To identify phishing indicators in email messages, classify emails based on risk level, and promote cybersecurity awareness through practical analysis and prevention guidelines.

**Tools Used**

* Email Samples
* Notepad
* Browser Tools
* Canva
* PDF Documentation

 **Activities Performed**

* Email Analysis
* Phishing Indicator Identification
* Email Classification
* Risk Assessment
* Security Awareness Documentation

 **Email Classifications**

* Safe Email
* Suspicious Email
* Phishing Email

 **Files Included**

* Phishing Email Detection and Awareness Report (PDF)
